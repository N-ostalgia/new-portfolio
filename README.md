# Aya Bouibauan – Developer Portfolio

Personal portfolio website showcasing my projects, skills, and experience as a Computer Science Engineering student at ENSAH, Morocco. Built with React, Vite, and React Flow.

🔗 **Live demo:** [https://github.com/N-ostalgia/new-portfolio]
---

## ✨ Features

- **Dark theme** with gold accents and custom cursor
- **Interactive project diagrams** (React Flow) – click any project screenshot to explore system architecture
- **Typewriter hero section** and scroll‑triggered animations
- **Responsive design** – works on desktop, tablet, and mobile
- **Downloadable CV** and contact form (powered by Formspree)

---

## 🛠️ Tech Stack

- **Frontend:** React 18, Vite, JavaScript (JSX)
- **Styling:** Custom CSS with CSS variables (no external frameworks)
- **Diagrams:** `@xyflow/react` (React Flow)
- **Icons:** Inline SVG
- **Hosting:** Netlify
